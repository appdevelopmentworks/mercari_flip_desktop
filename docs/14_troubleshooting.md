# トラブルシューティング

## keyring 関連
- 認証エラーが出る：OSの資格情報ストアの権限設定を確認
- macOS で保存失敗：キーチェーンの許可ダイアログを確認

## 起動エラー
- ModuleNotFoundError: PySide6：仮想環境の有効化と依存再インストール
- Qt platform plugin：環境変数 QT_PLUGIN_PATH を確認

## API 関連
- 401/403：APIキーの再設定と権限確認
- 429：レート制限のためリトライ間隔を調整

## SQLite 関連
- ロックエラー：多重起動を避ける
- パス不正：`data/` の位置と書き込み権限を確認
