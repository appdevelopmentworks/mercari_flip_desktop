# UI仕様（PySide6 / 3ペイン）

## 画面構成（メインウィンドウ）
- 左ペイン：商品リスト（検索・フィルタ・追加/編集/削除）
- 中央ペイン：仕入れ候補（Offer一覧、更新ボタン、ソート、最安表示）
- 右ペイン：配送・利益計算
  - 配送入力（縦/横/高さ/重量/資材費）
  - 推奨配送候補一覧（最安バッジ）
  - 売価入力（想定売価）
  - 相場入力（low/mid/high + メモ）
  - 計算結果（粗利/利益率/損益分岐/目標ライン）
  - 「計算結果を保存」ボタン

## メニュー
- 設定（APIキー/既定値/送料表編集）
- CSV入出力
- ログフォルダを開く（任意）

## 主要操作
- 商品選択 → 「候補更新」 → Offer一覧更新
- Offerの「最安」をワンクリックで利益計算に反映（原価として採用）
- 配送条件入力 → 「配送推定」自動更新（入力のたび or ボタン）
- 売価入力 → 利益計算更新（入力のたび or ボタン）

## バリデーション
- 数値は負数不可
- 寸法は 0〜（上限は適度に）
- shipping=NULL のOfferは警告表示（“送料不明”）
- 価格.com（Tavily）候補は confidence に応じて色/アイコンで注意喚起（MVPはテキストでもOK）

## 非同期表示
- 取得中はプログレス表示（スピナー/ステータスバー）
- 途中で失敗しても、成功分は表示
